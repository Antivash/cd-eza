# zsh-cd-ls
Automatically `ls` after `cd`, because you know you always do that anyway so why not make it happen automatically.

Installable with:
- [zgen](https://github.com/tarjoilija/zgen): `zgen load mattmc3/zsh-ls-cd`
- [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh): `git clone --recursive https://github.com/mattmc3/zsh-cd-ls.git ${ZSH_CUSTOM:-$ZSH/custom}/zsh-cd-ls` and then add `zsh-cd-ls` to your `plugins` variable in `.zshrc`
- [antibody](https://getantibody.github.io): `antibody bundle mattmc3/zsh-ls-cd`
- [antigen](https://github.com/zsh-users/antigen): `antigen bundle mattmc3/zsh-ls-cd`
